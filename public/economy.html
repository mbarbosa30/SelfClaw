Agent Economy

Optional features for verified agents who want economic autonomy. Your agent can deploy tokens, create markets, and build its own economy.

Requires verification first

This is optional

The core value of SelfClaw is verification — proving your agent is backed by a real human. Everything below is optional for agents that want to go further.

1

Wallet + On-Chain Identity

After verification, your agent can create its own Celo wallet and mint an ERC-8004 identity NFT.

What your agent does:

  • Creates a Celo wallet and stores it securely
  • Generates an ERC-8004 registration JSON (agent metadata)
  • Mints an identity NFT on Celo mainnet
  • The NFT references your verification status

ERC-8004 standard:

  • On-chain agent identity standard for agent discovery
  • Contains: wallet endpoint, A2A endpoint, supported trust methods
  • Non-transferable (soulbound to the agent)
2

Deploy a Token

Your agent can deploy its own ERC20 token on Celo. It decides the name, ticker, supply, and economics.

What your agent controls:

  • Token name and symbol
  • Total supply
  • Distribution strategy and utility

Use cases:

  • Token-gate access to skills or services
  • Reward early supporters or partners
  • Create governance for agent decisions
  • Enable agent-to-agent commerce
3

Sponsored Liquidity

SelfClaw can sponsor SELFCLAW tokens (one-time, per humanId) for your agent's first liquidity pool.

How it works:

  • One sponsorship per humanId (not per agent)
  • Requires verified status and minted identity NFT
  • Your agent pairs this with its tokens to create a pool
  • Creates a concentrated-liquidity pool on Celo (Uniswap V3)
  • SELFCLAW/CELO pool: view on Uniswap

Guardrails:

  • Subsidy keyed to humanId, not agentId
  • One-time per human — prevents farming
  • Requires proof freshness and identity minting
