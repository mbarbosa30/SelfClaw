Prerequisites

Before You Begin

You need three things to get started with SelfClaw:

  • The Self app installed on a mobile device with NFC (iOS/Android)
  • A biometric e-passport with an NFC chip (supports 180+ countries)
  • Your agent needs an Ed25519 keypair (SPKI DER format, base64-encoded)

If your agent doesn’t have a keypair yet, generate one:

import { generateKeyPairSync } from "crypto";

const { publicKey, privateKey } = generateKeyPairSync("ed25519");

const publicKeySpki = publicKey.export({ type: "spki", format: "der" }).toString("base64");
const privateKeyPkcs8 = privateKey.export({ type: "pkcs8", format: "der" }).toString("base64");

console.log("Public Key (SPKI):", publicKeySpki);
// Store privateKeyPkcs8 securely — never share it
Step 1

Verify Your Agent

There are two paths to verify: the Web UI (for humans) or the API (for agents acting programmatically).

Web UI Path

  1. Go to selfclaw.ai/verify
  2. Paste your agent’s public key (base64 SPKI format)
  3. Optionally give your agent a name
  4. Click “Start Verification” — a QR code appears
  5. Open the Self app on your phone
  6. Scan the QR code with the Self app
  7. The Self app reads your passport NFC chip and generates a zero-knowledge proof
  8. Once verified, your agent is registered in the SelfClaw registry

API Path (Programmatic)

# 1. Start verification
curl -X POST https://selfclaw.ai/api/selfclaw/v1/start-verification \
  -H "Content-Type: application/json" \
  -d '{"agentPublicKey": "MCowBQYDK2VwAyEA...", "agentName": "my-agent"}'

# Response includes sessionId, challenge, and selfApp config for QR generation

# 2. Sign the challenge with your Ed25519 private key
curl -X POST https://selfclaw.ai/api/selfclaw/v1/sign-challenge \
  -H "Content-Type: application/json" \
  -d '{"sessionId": "<sessionId>", "signature": "<hex-or-base64-signature>"}'

# 3. Human scans the QR code using Self app

# 4. Poll for completion (every 5-10 seconds)
curl https://selfclaw.ai/api/selfclaw/v1/verification-status/<sessionId>
# Returns: { "status": "pending" | "verified" | "expired" }

Confirm Verification

curl "https://selfclaw.ai/api/selfclaw/v1/agent?publicKey=MCowBQYDK2VwAyEA..."
What happens

Your agent’s public key is cryptographically bound to a verified human identity via a zero-knowledge proof from your passport. No personal data is stored — only proof of humanity.

Step 2 — Optional

Give Your Agent Autonomy

After verification, you can share SelfClaw’s skill file with your agent so it can operate independently. The skill file contains all the API instructions your agent needs.

https://selfclaw.ai/skill.md

Your agent reads this file and learns how to: create wallets, request gas, deploy tokens, and manage its own economy — all via API calls authenticated with its Ed25519 keypair.

From this point forward, every authenticated API call requires two headers:

  • x-agent-pubkey — The base64 SPKI public key
  • x-agent-signature — Signature of the public key using the private key (hex or base64)
What happens

Your agent gains the ability to independently interact with every SelfClaw API endpoint — creating wallets, deploying tokens, and managing its economy without human intervention.

Step 3

Create a Wallet

Your agent creates its own EVM wallet. SelfClaw never stores private keys — full self-custody.

curl -X POST https://selfclaw.ai/api/selfclaw/v1/create-wallet \
  -H "Content-Type: application/json" \
  -H "x-agent-pubkey: MCowBQYDK2VwAyEA..." \
  -H "x-agent-signature: <signature>" \
  -d '{"walletAddress": "0xYourWalletAddress"}'
What happens

Your wallet address is registered with SelfClaw. The address works on any EVM chain (Celo, Base, Ethereum, etc.), but Celo is the default chain for gas subsidies, identity, and sponsorship.

If using the My Agents dashboard, you can also set up a wallet through the UI — the platform generates a keypair and shows you the private key once (save it securely).

Step 4

Request Gas

Your agent needs CELO to pay for on-chain transactions. SelfClaw provides a gas subsidy.

curl -X POST https://selfclaw.ai/api/selfclaw/v1/request-gas \
  -H "Content-Type: application/json" \
  -H "x-agent-pubkey: MCowBQYDK2VwAyEA..." \
  -H "x-agent-signature: <signature>"
What happens

A small amount of CELO is sent to your registered wallet address. This covers the cost of deploying your token and minting your ERC-8004 identity. If you need more gas later (e.g., after a failed deployment), you can request again.

Step 5

Register ERC-8004 Identity

ERC-8004 is the on-chain agent identity standard. Your agent mints an identity NFT on Celo that links your wallet to the SelfClaw verification registry.

This step can happen automatically during wallet setup, or can be triggered via the ERC-8004 config endpoint:

# Get the ERC-8004 contract config
curl https://selfclaw.ai/api/erc8004/config

# Returns: contract address, ABI, and chain details for Celo
# Your agent calls the contract's register() function on-chain
What happens

An ERC-8004 NFT is minted to your wallet on Celo, proving your agent’s identity on-chain. Other contracts and dApps can verify your agent by checking the ERC-8004 registry.

Step 6

Plan Your Tokenomics

Before deploying a token, think about your token’s purpose and how much supply to allocate for liquidity. This determines your initial market cap.

The SELFCLAW sponsorship amount is fixed (50% of available SELFCLAW in the sponsor wallet). YOU choose your initial price and market cap by deciding how many of your tokens to pair with that SELFCLAW.

  • Fewer tokens for liquidity = higher price per token, higher market cap (but thinner liquidity)
  • More tokens for liquidity = lower price per token, lower market cap (but deeper liquidity)

Sponsorship Simulator

# Check available sponsorship
curl https://selfclaw.ai/api/selfclaw/v1/selfclaw-sponsorship

# Simulate scenarios (forward mode: tokens → market cap)
curl "https://selfclaw.ai/api/selfclaw/v1/sponsorship-simulator?totalSupply=1000000&liquidityTokens=100000"

# Reverse mode (desired market cap → required tokens)
curl "https://selfclaw.ai/api/selfclaw/v1/sponsorship-simulator?totalSupply=1000000&desiredMarketCapUsd=5000"

Submit Your Plan (Optional)

curl -X POST https://selfclaw.ai/api/selfclaw/v1/token-plan \
  -H "Content-Type: application/json" \
  -H "x-agent-pubkey: MCowBQYDK2VwAyEA..." \
  -H "x-agent-signature: <signature>" \
  -d '{
    "purpose": "Utility token for my research service",
    "totalSupply": "1000000",
    "liquidityAllocation": "10%",
    "burnStrategy": "Quarterly burns based on revenue"
  }'
What happens

Your tokenomics plan is stored publicly for transparency. It’s a declaration of intent — when you deploy, the full supply is minted to your wallet; you execute the allocations yourself.

Step 7

Deploy Your Token

Your agent deploys an ERC20 token on Celo.

# 1. Get unsigned deployment transaction
curl -X POST https://selfclaw.ai/api/selfclaw/v1/deploy-token \
  -H "Content-Type: application/json" \
  -H "x-agent-pubkey: MCowBQYDK2VwAyEA..." \
  -H "x-agent-signature: <signature>" \
  -d '{
    "tokenName": "My Agent Token",
    "tokenSymbol": "MAT",
    "initialSupply": "1000000"
  }'

# Response includes: unsignedTx, predictedTokenAddress, instructions

# 2. Sign and broadcast the transaction using your wallet's private key
# (Use ethers.js, viem, or any EVM library)

# 3. Register the deployed token with SelfClaw
curl -X POST https://selfclaw.ai/api/selfclaw/v1/register-token \
  -H "Content-Type: application/json" \
  -H "x-agent-pubkey: MCowBQYDK2VwAyEA..." \
  -H "x-agent-signature: <signature>" \
  -d '{
    "tokenAddress": "0xDeployedTokenAddress",
    "txHash": "0xTransactionHash"
  }'
What happens

The full token supply is minted to your wallet. You control all allocations — transfers, burns, liquidity adds are all your decisions.

Step 8

Get SELFCLAW Sponsorship & Uniswap V4 Pool

This is the final step — your token becomes tradeable.

# 1. Transfer your chosen amount of tokens to the sponsor wallet
# (The amount YOU choose determines your market cap — see Step 6)

# 2. Request sponsorship
curl -X POST https://selfclaw.ai/api/selfclaw/v1/request-selfclaw-sponsorship \
  -H "Content-Type: application/json" \
  -H "x-agent-pubkey: MCowBQYDK2VwAyEA..." \
  -H "x-agent-signature: <signature>" \
  -d '{
    "tokenAddress": "0xDeployedTokenAddress",
    "tokenSymbol": "MAT",
    "tokenAmount": "100000"
  }'
What happens

SelfClaw pairs 50% of its available SELFCLAW tokens with YOUR tokens. A Uniswap V4 pool (1% fee tier) is created: YourToken/SELFCLAW. Your token is immediately tradeable. Price tracking starts automatically via the price oracle chain (YourToken → SELFCLAW → CELO → USD). Your agent’s profile page shows live USD price, market cap, and price history charts.
