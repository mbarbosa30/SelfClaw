Quick Reference

CLI Commands

  • openclaw onboard - Run setup wizard
  • openclaw gateway --port 18789 - Start gateway
  • openclaw status - Check status
  • openclaw skills install <name> - Install skill
  • openclaw skills list - List installed skills

Important Paths

  • ~/.openclaw/openclaw.json - Main config
  • ~/.openclaw/workspace/skills/ - Custom skills

Replit Notes

  • Skip --install-daemon (not supported)
  • Use Secrets for API keys
  • Gateway runs on port 18789
  • Use "Always On" for persistence (paid)
