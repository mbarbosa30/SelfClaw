OPENCLAW

REPLIT EDITION

WHAT IS THIS?

OpenClaw is an open-source personal AI assistant framework. It connects to messaging apps (Telegram, Discord, Slack) and uses AI models (Claude, GPT) to automate tasks like email management, calendar scheduling, web browsing, and custom workflows.

This platform lets you experiment with OpenClaw in a sandboxed cloud environment without local setup complexity.

HOW IT WORKS

  1. Setup — Initialize directories and configuration files
  2. Configure — Add your AI provider keys and bot tokens
  3. Install — Get the OpenClaw CLI tool
  4. Run — Start the gateway to handle AI communications
  5. Connect — Message your bot via Telegram/Discord/Slack

GETTING STARTED

01

Initialize Environment

Creates ~/.openclaw directory, default configuration, and sample skill.

02

Add API Keys

Go to Replit Secrets (Tools → Secrets) and add your keys.

ANTHROPIC_API_KEY —

03

Install OpenClaw CLI

Installs the openclaw command globally via npm.

04

Run Onboarding (Shell)

Open the Shell tab and run:

openclaw onboard

Skip --install-daemon (not supported on Replit)

05

Start Gateway

Launches the AI gateway on port 18789.

CONFIGURATION

Edit ~/.openclaw/openclaw.json directly. Changes take effect on gateway restart.

REFERENCE

CLI COMMANDS

  • openclaw onboard — Setup wizard
  • openclaw gateway --port 18789 — Start gateway
  • openclaw status — Check status
  • openclaw skills install <name> — Install skill
  • openclaw skills list — List skills

FILE PATHS

  • ~/.openclaw/openclaw.json — Main config
  • ~/.openclaw/workspace/skills/ — Custom skills

USE CASES

  • Email and calendar automation
  • Content creation pipelines
