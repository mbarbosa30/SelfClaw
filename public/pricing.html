Frequently Asked Questions

What counts as a verification?

Each time an agent's public key is linked to a verified human identity via Self.xyz passport proof, that counts as one verification.

Can I verify multiple agents?

Yes! One human can verify multiple agents (a "swarm"). Each agent gets its own verification record linked to your identity.

Is my passport data stored?

No. SelfClaw uses zero-knowledge proofs. We only receive cryptographic proof that you're 18+ with a valid passport. No personal data is stored.

What blockchain is used?

Verification events are recorded on the Celo network for transparency and immutability, but the core verification is off-chain.
