SelfClaw API Documentation

Integrate agent verification into your application. Verify that AI agents are backed by real humans.

Quick Start

Base URL: https://selfclaw.ai/api/selfclaw/v1

Lookup Agent

Check if an agent is verified and retrieve its verification status.

GET /agent/{identifier}

Lookup an agent by public key or key hash. Returns verification status and human association.

Parameter Type Description
identifier string required Agent public key or 16-character key hash 
// Response
{
  "verified": true,
  "agentKeyHash": "2a8eff8fcc16977c",
  "humanId": "a1b2c3d4e5f6g7h8",
  "verifiedAt": "2026-02-03T12:00:00Z",
  "swarmSize": 3,
  "signatureVerified": true
}

Start Verification

Initiate a new verification session for an agent.

POST /start-verification

Creates a verification session and returns the Self.xyz QR code configuration.

Parameter Type Description
agentPublicKey string required The agent's public key to verify
agentName string optional Human-readable name for the agent
signature string optional Ed25519 signature of the challenge to prove key ownership 
// Request
POST /api/selfclaw/v1/start-verification
{
  "agentPublicKey": "ed25519:abc123...",
  "agentName": "ResearchBot"
}

// Response
{
  "success": true,
  "sessionId": "uuid-v4",
  "agentKeyHash": "2a8eff8fcc16977c",
  "challenge": "{...}",
  "selfApp": { /* Self.xyz QR config */ }
}

Get Human's Agents

List all agents verified by a specific human (swarm lookup).

GET /human/{humanId}

Returns all agents associated with a humanId.

// Response
{
  "humanId": "a1b2c3d4e5f6g7h8",
  "agents": [
    {
      "agentKeyHash": "2a8eff8fcc16977c",
      "agentName": "ResearchBot",
      "verifiedAt": "2026-02-03T12:00:00Z"
    },
    {
      "agentKeyHash": "9f8e7d6c5b4a3210",
      "agentName": "TradingBot",
      "verifiedAt": "2026-02-02T10:00:00Z"
    }
  ],
  "swarmSize": 2
}

Registry Statistics

Get public statistics about the verification registry.

GET /stats

Returns aggregate statistics about verified agents.

// Response
{
  "totalVerifiedAgents": 1234,
  "uniqueHumans": 567,
  "verificationsLast24h": 42
}

Error Codes

Code Description
400 Bad request - missing or invalid parameters
404 Agent or human not found
429 Rate limit exceeded (100 requests/15 min)
500 Internal server error
